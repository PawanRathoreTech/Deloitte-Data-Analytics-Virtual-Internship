# Deloitte Australia Data Analytics Virtual Internship

## 📌 Overview
This project is part of the Deloitte Australia Data Analytics Job Simulation on Forage. It involves analyzing operational and employee data to derive meaningful insights.

## 🛠 Tools Used
- Microsoft Excel
- Tableau

## 📊 Tasks Performed
- Created an interactive dashboard to analyze downtime across factories and machines
- Identified high downtime areas using data visualization
- Classified employee equality scores into categories (Fair, Unfair, Highly Discriminative)

## 🔍 Key Insights
- The Seiko factory had the highest downtime
- Laser machines contributed most to operational inefficiencies
- Some roles showed signs of pay inequality

## 📁 Files Included
- Excel file with classification (Task 2)
- Tableau dashboard screenshot
- Completion certificate

## 🚀 Outcome
This project helped in understanding real-world data analysis, visualization, and decision-making processes.
